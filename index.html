About

Hi, I'm Matt Colman and I'm a Front End Developer. My main weapon of choice is javascript. I love javascript! I love lamp. I'm passionate about clever code. Performant, light-weight, readable, extendable. Sometimes there's a compromise, but I always aim to achieve them all.

I choose to be a front end developer because I'm inspired by seeing great design come to life.

I strive to create lag-free, jank-free, 60fps web apps and games. People really notice when something moves beautifully on the web.

I'm canvas obsessed. I've made games in canvas using CreateJS and Phaser. More recently I've been making apps with React. Currently I'm excited by ES6 and functional javascript. Basically if it's javascript...

Tutoring

Coding is quickly popping up as an optional subject in many schools and a mandatory subject in some. It's becoming very obvious that this is a necessary skill for the current generation of kids.

The internet isn't going anywhere!

Web developers and software engineers of any kind are becoming more and more in demand.

It's never too early to start learning, and I'm here to help!

I'm available for tutoring school students of all ages. I can teach students how to love coding by making games for the web or mobile apps. I can help students taking on coding subjects at school. I can teach anyone how to make a website with React.

Fantasy Sports website (React)

Worked in a team to build a Fantasy Sports single page web app using React and Redux.

Tech used:

  • React
  • Redux
  • Webpack
  • Greensock
  • NPM
  • JSS
  • Flowtype
  • ESLint
  • Jest

Main focuses:

  • Composition. Small, reusable, stateless components.
  • Introducing subtle yet effective animations using css and greensock
  • App stability using Flowtype, ESlint and Jest to minimize regressions.
